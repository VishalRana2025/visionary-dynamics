# IT Consulting Header Demo

This small project contains a responsive header component built with React + Tailwind CSS and scaffolded for Vite.

Quick start

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

Files added

- `index.html`
- `package.json`
- `postcss.config.cjs`, `tailwind.config.cjs`
- `src/index.css`, `src/main.jsx`, `src/App.jsx`, `src/components/Header.jsx`
